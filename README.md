# Projeto-usf-POO
Aplicação desenvolvida em Python utilizando Programação Orientada a Objetos (POO) com interface web criada em Streamlit.

O sistema simula o funcionamento de um computador, permitindo criar um modelo personalizado e executar ações como ligar, desligar, instalar programas, visualizar programas instalados e formatar o disco.

Funcionalidades

Criar um computador personalizado

Ligar e desligar o computador

Instalar programas

Listar programas instalados

Formatar disco

Exibir histórico de ações

Interface web interativa

Conceitos Aplicados

Programação Orientada a Objetos

Encapsulamento (atributos privados)

Getters e Setters

Manipulação de estado

Interface web com Streamlit

Session State
